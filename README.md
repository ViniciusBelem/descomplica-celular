## 🚀 Visão Geral

A Versão 3 (V3) marca o pivot total de um protótipo estático para uma **Plataforma de Decisão de Alta Fidelidade**. Construída sobre uma arquitetura moderna e escalável, a aplicação utiliza algoritmos de match dinâmicos para conectar o orçamento e as necessidades do usuário ao smartphone ideal.

### 💎 Funcionalidades de Elite

- **🧠 Consultor Neural:** Fluxo interativo de 4 passos que calcula a afinidade (Match Score) em tempo real.
- **📱 Smart Catalog:** Vitrine expansiva com filtros multi-facetados, ordenação tática e insights técnicos (Câmera, Bateria, CPU) acessíveis via hover.
- **📚 Saved Collection:** Biblioteca persistente para gestão de interesses e retenção de intenção de compra.
- **⚖️ Duelo Técnico:** Comparador profissional de 3 slots que revela a disparidade de hardware e valor.
- **🛡️ Admin Shield:** Painel de controle blindado para gestão de inventário e links de afiliados com Row-Level Security (RLS).

## 🛠️ Stack Tecnológica

- **Core:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand) (Persistência Atômica)
- **Database & Auth:** [Supabase](https://supabase.com/) (PostgreSQL real-time)
- **Interface:** [Tailwind CSS](https://tailwindcss.com/) com estética **Glassmorphism Premium**
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Protocolo de Engenharia:** Desenvolvido sob a matriz **A.E.S. 4.1 (Algorithmic Evolutionary System)** — garantindo integridade arquitetural e precisão cirúrgica em cada commit.

## ⚙️ Instalação e Execução

Para rodar o motor localmente:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/ViniciusBelem/descomplica-celular.git
   ```

2. **Instale as dependências:**
   ```bash
   cd descomplica-v3
   npm install
   ```

3. **Configure as Variáveis de Ambiente:**
   Crie um arquivo `.env` dentro da pasta `descomplica-v3/`:
   ```env
   VITE_SUPABASE_URL=sua_url_aqui
   VITE_SUPABASE_ANON_KEY=sua_chave_aqui
   ```

4. **Inicie o Servidor:**
   ```bash
   npm run dev
   ```
