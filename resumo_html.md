# Resumo: Fundamentos do HTML

## O que é o HTML?
HTML (HyperText Markup Language) não é uma linguagem de programação, pois não possui lógica (como if/else). É uma **Linguagem de Marcação** utilizada para estruturar e dar significado ao conteúdo de uma página web.

## Glossário de Principais Tags
* `<h1>` a `<h6>`: Definem os títulos e subtítulos da página, criando uma hierarquia.
* `<p>`: Define um parágrafo de texto regular.
* `<a>`: Cria um hiperlink para navegar para outras páginas, utilizando o atributo `href`.
* `<img>`: Insere uma imagem na página. É uma tag singular e exige o atributo `src` e `alt`.
* `<ul>` e `<ol>`: Criam listas não ordenadas e ordenadas. Seus itens são definidos pela tag `<li>`.

## A importância da tag `<div>`
A tag `<div>` (division) é um contêiner genérico. Ela não possui valor semântico próprio, mas é fundamental para agrupar e aninhar outros elementos para organização de layout.
